Course Discription

This class is designed for 12 + years old kids. Students will learn the syntax of the Java programming language, which can be used for programming most types of applications. And for many years the Java programming language has been one of the world‚Äôs most popular computer languages. My goal is to help students learn to program in the most popular language in the world: Java. We start from an introduction to Java using greenfoot to learn OOP concepts like class, object, inheritance, parent-child relationship of the class then the second part of the class, we will learn how to write programs that have Graphic User Interface by writing the Tic-Tac-Toe and Ping-Pong games. then we will revisit java sytax in a more specific way like variable, reference, eight fundamental data types, decisions controls flew, loops. we also intruduce three to five data structures like arrays , ArrayList, LinkedList, sets and maps. we will also learn an important algorithm in problem solving :Recursion At last we we move to different area which is interacted website design using HTML, CSS,JAVAScript

Teaching Method

I will use code-driven method throughout the course. I will code along with the students side by side every step the way. Students will engage themself in writing codes to build games and apps in a fun interacted way. They will use all the concepts they learned to build games which they enjoy playing.
